# How to use the application

## Keys
|Key|Action|
|-|-|
|`up/down arrow`|Navigate through lists|
|`ctrl+q/ctrl+e`|Switch between Downloads and Guides|
|`Enter`|Selects Option|
|`ctrl+c`|quits application at once|
